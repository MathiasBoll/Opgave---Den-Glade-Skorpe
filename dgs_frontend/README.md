# Den Glade Skorpe — Frontend

React 18 + Vite frontend for Den Glade Skorpe pizzeria.

For full setup instructions, endpoints, feature checklist and test credentials, see the root [`README.md`](../README.md).

## Quick start

```bash
npm install
npm run dev
```

Frontend runs on **http://localhost:5174** (backend must be running on port 3042).
