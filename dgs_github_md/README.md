# Den Glade Skorpe - GitHub issues og projektfiler

Disse markdown-filer er lavet ud fra opgavebeskrivelsen til **Den Glade Skorpe**.

Jeg prøvede først at oprette issues direkte i GitHub-repoet `MathiasBoll/Opgave---Den-Glade-Skorpe`, men GitHub-integrationen gav fejlen **Resource not accessible by integration (403)**. Derfor er filerne lavet som `.md`, så du selv kan uploade dem eller copy/paste dem ind som GitHub Issues.

## Sådan bruger du filerne

1. Gå ind på dit GitHub-repo.
2. Gå til **Issues**.
3. Tryk **New issue**.
4. Brug titlen fra den relevante `.md`-fil.
5. Copy/paste indholdet fra filen som issue body.
6. Gentag for de issues, du vil have på GitHub.

Derefter:

8. Employees side
9. Contact formular
10. Basket side

15-20. Tilvalg, hvis der er tid
