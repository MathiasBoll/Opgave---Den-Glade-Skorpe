# [Mandatory] Set up frontend project, routing and folder structure

## Goal
Create the frontend foundation with a modern framework, routing and an exam-friendly folder structure.

## Tasks
- [ ] Set up React/Vite or another modern frontend framework.
- [ ] Install routing.
- [ ] Create these routes: `/`, `/dish/:id`, `/employees`, `/contact`, `/basket`, `/backoffice`, `/backoffice/employees`.
- [ ] Create a 404 route.
- [ ] Create folders for `components`, `pages`, `services`, `utils`, `assets` and styling.
- [ ] Add provided assets: `logo.png`, `headerImg.png`, `basket_icon.png`.
- [ ] Make sure the app runs locally.
- [ ] Add `.gitignore` so `node_modules` is not committed.

## Acceptance criteria
- [ ] All required routes render without crashing.
- [ ] The folder structure is clear and easy to explain.
- [ ] Project can be started locally.

## Estimate
3-4 hours.
