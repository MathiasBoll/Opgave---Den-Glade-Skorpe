# [Mandatory] Build front page with dishes and category filter

## Goal
Build the front page `/` with intro text, categories and all dishes from the API.

## Tasks
- [ ] Fetch categories from API.
- [ ] Fetch dishes from API.
- [ ] Show loading state while data loads.
- [ ] Show error message if fetch fails.
- [ ] Display all dishes as cards.
- [ ] Each dish card shows image and name, plus price/description if suitable.
- [ ] Add category filter section: “Vælg kategori”.
- [ ] Filter dishes dynamically without page reload.
- [ ] Add active styling for selected category.
- [ ] Dish cards link to `/dish/:id`.
- [ ] Add intro text about Den Glade Skorpe.

## Acceptance criteria
- [ ] Front page shows all dishes.
- [ ] Category filter works.
- [ ] Clicking a dish opens its detail page.
- [ ] Loading/error states exist.
