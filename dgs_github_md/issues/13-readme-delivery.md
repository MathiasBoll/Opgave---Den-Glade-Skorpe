## Goal
Make the project easy for teacher/examiner to run and check.

## README must include
- [ ] Project name.
- [ ] Short description.
- [ ] Frontend setup instructions.
- [ ] Backend setup instructions.
- [ ] Commands to install dependencies.
- [ ] Commands to start frontend.
- [ ] Commands to start backend.
- [ ] Backend URL: `http://localhost:3042`.
- [ ] Required `.env.local` info if relevant.
- [ ] Completed features.
- [ ] Known missing features.
- [ ] Login info if authentication is used.

## Delivery checks
- [ ] `node_modules` is not included.
- [ ] Project runs after fresh install.
- [ ] Report is in root as `.md` or `.pdf`.
- [ ] README is in root.
- [ ] No edits after deadline unless in a copy.

## Acceptance criteria
- [ ] A new person can run the project from README.
- [ ] Delivery contains source code, README and report.
