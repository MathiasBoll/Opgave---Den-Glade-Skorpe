# [Mandatory] Build contact form with validation and POST to server

## Goal
Build `/contact` with a validated contact form that sends data to the backend.

## Fields
- name/navn
- subject/emne
- message/besked

## Tasks
- [ ] Create form layout matching the design.
- [ ] Validate that all fields are filled.
- [ ] Show user-friendly validation messages.
- [ ] POST data to server using `/message`.
- [ ] Show loading state while submitting.
- [ ] Show success message: “Tak for din besked!”
- [ ] Show error message if submit fails.
- [ ] Clear form after successful submit.

## Acceptance criteria
- [ ] Empty form cannot be submitted.
- [ ] Valid form sends message to backend.
- [ ] User gets success/error feedback.

## Estimate
4-6 hours.
