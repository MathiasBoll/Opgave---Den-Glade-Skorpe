## What to upload/copy

### Best single file
Use:

`ALL_EXISTING_ISSUES_EXPANDED.md`

This contains every existing issue expanded in one long file.

### Individual files
Use files in:

`issues_expanded/`

Each file matches one existing GitHub issue.

### Paste as comments
Use files in:

`github_copy_paste_comments/`

These are formatted so you can paste them as comments on the existing GitHub issues without replacing the original body.

### Coverage audit
Use:

`PDF_COVERAGE_AUDIT.md`

This checks the PDF requirements against the issue plan.

### Delivery checklist
Use:

`checklists/PROJECT_CHECKLIST_EXPANDED.md`

Use this before handing in the assignment.
